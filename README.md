# dotconfig

All configuration files located under ~/.config (or not) that I would like to save.

## yazi

Actually, yazi configuration files located under `$env:APPDATA\yazi\config` (on Windows). You will have to copy these files.

### Themes / Flavors

Preferred themes:

- https://github.com/yazi-rs/flavors/tree/main/dracula.yazi
- https://github.com/yazi-rs/flavors/tree/main/catppuccin-macchiato.yazi
- https://github.com/dangooddd/kanagawa.yazi
- 👑 https://github.com/dangooddd/kanagawa.yazi
- 👑 https://github.com/matt-dong-123/gruvbox-material.yazi
- https://github.com/hankertrix/bluloco-yazi/tree/main/bluloco-dark.yazi

Installation:

```shell
ya pkg add yazi-rs/flavors:dracula
ya pkg add yazi-rs/flavors:catppuccin-macchiato
ya pkg add dangooddd/kanagawa
ya pkg add matt-dong-123/gruvbox-material
ya pkg add AdithyanA2005/nord
ya pkg add Chromium-3-Oxide/everforest-medium
```
